Um curso que explora os beneficios de utilizar o Git como ferramenta de controle de versao para projetos em qualquer linguagem, em qualquer plataforma.

Principais caracteristicas:

  • Funciona de maneira distribuida
  • Permite a edicao concorrente de arquivos do projeto
  • Nao depende de uma conexao ativa com um servidor
